# WellNest 参加者向けユーザーマニュアル

WellNest 研究にご参加いただき、ありがとうございます。

このガイドでは、Android アプリのインストール、初期設定、アンケートへの回答、アプリが正しく動作しない場合の対応方法を説明します。研究参加者の方に向けた手順書ですので、順番に進めてください。

本研究は、東京大学 空間情報科学研究センターの研究チームによって実施されています。本研究では、産後のお母さまのウェルビーイングをよりよく理解するために、Passive Mobile Sensing 技術を活用できるかを検討します。

本研究には、以下の期間が含まれます。

- アプリに慣れていただくための約 1 週間のパイロット期間
- パイロット期間後に始まる 12 週間の正式なデータ収集期間

## はじめに

1. [アプリをインストールする]({{ '/ja/android/setup/' | relative_url }})  
   このようなリンクは、タップまたはクリックすると対応する章に移動できます。下にある紫色のリンクも同じように使えます。
   {: .clickable-note }

   リンクを開いたとき、正しいページには移動しているのに表示位置がずれている場合は、いったん前のページに戻り、同じ紫色のリンクをもう一度タップまたはクリックしてください。
   {: .link-jump-note }

   Firebase App Distribution の招待メールから、**WellNest** と **WellNest Health** の両方をインストールします。

2. [権限を許可する]({{ '/ja/android/permissions/' | relative_url }})  
   WellNest を開き、権限設定ページの各項目を完了します。

3. [事前アンケートに回答する]({{ '/ja/android/surveys/#pre-study-survey' | relative_url }})  
   初期設定後、**Join Study** をタップして事前アンケートに回答します。

## 時間がない場合

重要な設定と研究期間中の作業を確認するには、[クイックチェックリスト]({{ '/ja/android/checklist/' | relative_url }})をご利用ください。

## お困りの場合

研究チームまでメールでお問い合わせください。
