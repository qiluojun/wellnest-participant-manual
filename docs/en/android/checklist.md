# Quick Checklist

Use this checklist when you are short on time or want to confirm that everything is on track.

## Before You Start

- [ ] Installed both apps: **WellNest** and **WellNest Health**. See [Install the apps]({{ '/en/android/setup/' | relative_url }}).
- [ ] Granted all permissions. See [Grant permissions]({{ '/en/android/permissions/' | relative_url }}).
- [ ] Confirmed that the red exclamation mark has disappeared. See [How to confirm permissions are complete]({{ '/en/android/permissions/#how-to-confirm-permissions-are-complete' | relative_url }}).
- [ ] Allowed notifications. See [Notification settings]({{ '/en/android/permissions/#notification-settings' | relative_url }}).
- [ ] Completed the Pre-Study Survey. See [Pre-Study Survey]({{ '/en/android/surveys/#pre-study-survey' | relative_url }}).
- [ ] Confirmed WellNest is running in the background. See [Keeping the app running in the background]({{ '/en/android/settings/#keeping-the-app-running-in-the-background' | relative_url }}).
- [ ] Uninstalled WellNest Health only after permission setup was complete. See [When can I uninstall WellNest Health?]({{ '/en/android/permissions/#when-can-i-uninstall-wellnest-health' | relative_url }}).

## Every Day

- [ ] Morning Survey submitted between **6:00-10:00 am**. The app will send a reminder. See [Daily Surveys]({{ '/en/android/surveys/#daily-surveys' | relative_url }}).
- [ ] Evening Survey submitted between **8:00 pm-12:00 am**. The app will send a reminder. See [Daily Surveys]({{ '/en/android/surveys/#daily-surveys' | relative_url }}).
- [ ] WellNest is still running in the background. See [Keeping the app running in the background]({{ '/en/android/settings/#keeping-the-app-running-in-the-background' | relative_url }}).

## Every Monday

- [ ] Weekly Survey completed. If missed, go to **Help screen** -> **Weekly Survey**. See [Weekly Survey]({{ '/en/android/surveys/#weekly-survey' | relative_url }}).

## If Something Looks Wrong

- [ ] Red exclamation mark on Permissions page -> re-check the missing item. See [Permissions]({{ '/en/android/permissions/' | relative_url }}).
- [ ] App was closed -> relaunch the app. See [Keeping the app running in the background]({{ '/en/android/settings/#keeping-the-app-running-in-the-background' | relative_url }}).
- [ ] Upload failed -> open the **Data Upload screen** and tap the upload button. See [Manual data upload]({{ '/en/android/settings/#manual-data-upload' | relative_url }}).
- [ ] Anything else -> email [study-support@mcl.iis.u-tokyo.ac.jp](mailto:study-support@mcl.iis.u-tokyo.ac.jp). See [I need help]({{ '/en/android/troubleshooting/#i-need-help' | relative_url }}).
